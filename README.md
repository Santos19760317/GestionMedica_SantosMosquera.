# Sistema de Gestión Médica - Módulo de Registro de Pacientes

Este proyecto forma parte de la evidencia **GA7-220501096-AA3-EV02** para el programa de formación del SENA. Consiste en la codificación de una interfaz funcional para la captura de datos de pacientes.

## 🚀 Tecnologías Utilizadas
* **Lenguaje:** HTML5 y CSS3 para la estructura y diseño de la interfaz.
* **Entorno de Desarrollo (IDE):** IntelliJ IDEA 2025.3.3.
* **Control de Versiones:** Git y GitHub.
* **Stack Tecnológico:** Java (JDK 25) y Maven.

## 🛠️ Características del Módulo
El formulario incluye validaciones nativas para garantizar la integridad de los datos antes de ser procesados:
1. **Validación de Nombre:** Campo obligatorio con longitud mínima.
2. **Validación de Cédula:** Restricción para aceptar únicamente valores numéricos.
3. **Validación de Correo:** Validación de formato estándar (requiere @ y dominio).
4. **Validación de Fecha:** Selector de calendario para la fecha de registro.

## 📝 Estándares de Codificación
Se aplicaron los siguientes estándares sugeridos por el instructor:
* Uso de comentarios de autoría y descripción en el código fuente.
* Nombramiento de IDs y clases siguiendo la convención `lowerCamelCase`.
* Estructura de proyecto organizada mediante Maven.
